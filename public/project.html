Project Description

This project focuses on developing a machine learning algorithm to predict COPD exacerbations using a combination of clinical data, wearable device outputs, and patient-reported outcomes.

The goal is to create a predictive model that can alert clinicians to potential exacerbations 48-72 hours before they occur, allowing for proactive intervention and reduced hospitalizations.

Key Objectives:

  • Analyze historical clinical data from 500+ COPD patients
  • Integrate real-time wearable data (spirometry, activity levels)
  • Develop and validate prediction algorithm
  • Create clinician dashboard for real-time monitoring
